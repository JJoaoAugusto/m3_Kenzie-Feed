# m3_Kenzie-Feed
Desafio Front-End em grupo que simula um site de notícias onde usuários podem fazer o CRUD de uma notícia, visualizar as notícias de outros usuários com sistema de likes e deslikes e também fazer registro/login de uma conta . Projeto feito em React, abordando renderizações condicionais, manipulação de paginas, listagens e consumo de API.

Consiste no Projeto Final do Terceiro Módulo do Curso Full-Stack da Kenzie Academy Brasil.

https://kenzie-feed-zeta.vercel.app/
